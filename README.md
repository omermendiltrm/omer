# مجوهرات القوصرة - Al-Qusra Jewelry

موقع متقدم لمتجر المجوهرات مع عرض الأسعار المباشرة للذهب والفضة والعملات الرقمية والعملات الأجنبية.

## 🎨 المميزات

- ✨ **تصميم فاخر** بألوان ذهبية وسوداء
- 📊 **أسعار مباشرة** من TradingView
- 📱 **تطبيق ويب تقدمي** (PWA)
- 📞 **معلومات الاتصال** الكاملة
- 🌐 **دعم اللغة العربية** والإنجليزية
- 📈 **تكر الأسعار** المتحرك
